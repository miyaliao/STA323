Assignment 1 includes PySpark-based data cleaning and exploratory analysis, biological sequence processing, and LLM-assisted system design. # Project1 - PySpark, SQL & Survival Analysis

This project explores data analysis workflows using PySpark, MySQL, LLM-assisted SQL generation, and survival analysis. It analyzes diabetes health indicators to identify trends across different diabetes classes, showing that higher diabetes severity is associated with higher HighBP prevalence, higher average BMI, and lower physical activity. It also performs customer churn survival analysis on the IBM Telco Customer Churn dataset by using tenure as survival time and churn status as the event variable. The analysis includes Kaplan-Meier survival curves, log-rank tests, Cox Proportional Hazards modeling, and Log-Logistic AFT modeling to understand customer retention patterns and churn risk factors. In addition, this project evaluates LLM-generated Text-to-SQL queries and highlights typical failure cases such as incorrect column names, wrong aggregation logic, and invalid categorical assumptions. The final report was deployed through GitHub Pages with a custom domain to present the full analysis process and results.
